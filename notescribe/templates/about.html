Welcome to Notescribe

Who are we?

We are an open source project that aims to make music transcription easy.

Contributors

  • Satvik Reddy
  • Niels Voss
  • Soham Patil

Used Services and Libraries

  • Flask: Provides user with web pages
  • Pytorch: Machine learning models
  • Pydub: Converting audio to WAV format
  • LilyPond: Rendering midi as images
  • FPDF for Python: Packaging images as PDF
  • AWS S3 and boto3: Asset storage and upload management
  • Docker: Deployment configuration
  • AWS EC2: Website hosting

How it works

The user uploads a file to to a website. That file is processed and converted to WAV format. The newly created WAV file is processed by a machine learning algorithm, which generates a MIDI file.

The MIDI file is then converted to the intermediate LilyPond (.ly) format using midi2ly, and is then rendered as collection of images.

These images are packaged into a PDF document using FPDF. The PDF document, MIDI file, and generated images are stored in AWS S3 and their URLs are recorded in a json file. That json file is also uploaded to S3.

The user is sent back a URL that points ot the json file in S3, and client side JavaScript renders the images and provides links to download the PDF and MIDI.
